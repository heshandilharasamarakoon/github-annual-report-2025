# GitHub Annual Report 2025 - Frontend

React frontend with Vite, Tailwind CSS, and Framer Motion.

## Setup

1. Install dependencies:
```bash
bun install
```

2. Run the development server:
```bash
bun run dev
```

Frontend will start at http://localhost:5173

## Features

- GitHub OAuth authentication
- Real-time data fetching from GitHub GraphQL API
- Animated slide deck experience (Spotify Wrapped style)
- Interactive data visualizations:
  - Contribution heatmap
  - Radar chart for developer capabilities
  - Language distribution
  - Stats overview
- Dark minimalist theme with neon green accents

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Zustand (state management)
- Recharts (radar chart)
- react-activity-calendar (heatmap)
- Shadcn UI components
